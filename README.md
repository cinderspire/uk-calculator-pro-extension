# UK Calculator Pro - Browser Extension

Quick UK tax, salary, VAT and unit calculations right in your browser toolbar. Updated for the 2025/26 tax year.

## Features

### 1. Quick Tax Calculator
Calculate your take-home pay instantly. Enter your annual salary and see your monthly take-home after income tax and National Insurance.

### 2. VAT Calculator
Add or remove VAT at 20%, 5% or 0% with one click. Right-click any number on a webpage to calculate VAT.

### 3. Unit Converter
Convert between common UK/metric units:
- KG ↔ Stone & Pounds
- CM ↔ Inches
- Miles ↔ KM
- °C ↔ °F
- Litres ↔ Gallons

### 4. Bank Holiday Countdown
See how many days until the next UK bank holiday with a full list of remaining 2026 holidays.

### 5. Right-Click Context Menu
Select any number on a webpage and right-click to:
- Add 20% VAT
- Remove VAT
- Convert units (kg/lbs/cm/inches)

## Installation

### Chrome Web Store
[Install from Chrome Web Store](https://chrome.google.com/webstore)

### Microsoft Edge Add-ons
[Install from Edge Add-ons](https://microsoftedge.microsoft.com/addons)

### Manual Installation (Developer Mode)
1. Download this repository
2. Open `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select the extension folder

## UK Tax Rates (2025/26)
- Personal Allowance: £12,570
- Basic Rate: 20% (£12,571 - £50,270)
- Higher Rate: 40% (£50,271 - £125,140)
- Additional Rate: 45% (over £125,140)
- Employee NI: 8% / 2%
